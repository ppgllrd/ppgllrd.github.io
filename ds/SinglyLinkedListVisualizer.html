Singly Linked List

A singly linked list stores a sequence of values as nodes connected by pointers. Each node contains an element field and a p_next field.

The list structure contains only p_first, a pointer to the first node, or NULL when the list is empty.

Operations and asymptotic complexity

  • Append: O(n), because traversal starts at p_first.
  • Prepend: O(1).
  • Insert: O(n).
  • Sorted Insert: O(n). It uses p_previous and p_current to insert in ascending order. The button is disabled if the current list is not sorted ascending.
  • Get, Set, Delete: O(n).

The visualization shows how links and temporary pointers change while each algorithm progresses.
