SortedArraySet

A Java SortedArraySet stores the sequence in a heap allocated array of references elements. The object also tracks size. The capacity is implicitly the length of the internal array.

  • Append: ensure capacity, write the object reference at elements[size], then increment size. Complexity: amortized O(1); O(n) in the occasional worst case when reallocation occurs.
  • Prepend: insert at index 0, shifting all current references one position to the right. Complexity: O(n).
  • Insert: shift references from the insertion point to the right, write the new reference, then increment size. Complexity: O(n) in the worst case.
  • Get: direct access to elements[index]. Complexity: O(1).
  • Set: replace the reference at the selected index. Complexity: O(1).
  • Delete: shift elements to the left and explicitly set the last position to null to allow Java's Garbage Collector to free the memory. Complexity: O(n).
  • Clear: iterates the array setting all references to null and sets size to 0. Complexity: O(n) (in standard Java implementation).
  • Ensure capacity: when full, Arrays.copyOf allocates a larger array and copies the references over. Complexity: O(n).
