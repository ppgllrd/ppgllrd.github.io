ArrayStack<T>

An ArrayStack stores references in Java array elements. The field size is the number of stack elements and the first free position. The top element is at elements[size - 1].

  • push: ensure capacity, store at elements[size], then increment size.
  • top: return elements[size - 1] without removing it.
  • pop: decrement size and set elements[size] to null.
  • clear: set used references to null and leave the allocated array in place.

Complexity summary

  • push: amortized O(1), O(n) when resizing.
  • top, pop, isEmpty, size: O(1).
  • clear: O(n).
