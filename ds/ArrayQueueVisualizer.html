ArrayQueue<T>

This visualizer follows the supplied ArrayQueue implementation: DEFAULT_INITIAL_CAPACITY = 16, fields elements, first, last, and size, and circular advancement with advance(index) = (index + 1) % elements.length.

  • Constructor / clear: empty queue state uses first = 0 and last = elements.length - 1.
  • enqueue: calls ensureCapacity(), then last = advance(last), stores into elements[last], and increments size.
  • dequeue: sets elements[first] = null, advances first, and decrements size.
  • ensureCapacity: when full, allocates a doubled array and copies the queue from first to last into positions 0..size-1.
