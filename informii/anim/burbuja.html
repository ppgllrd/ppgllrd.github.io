El algoritmo de la burbuja ordena un array mediante un proceso iterativo que utiliza un índice denominado tam. Este índice comienza apuntando al último elemento del array y disminuye con cada iteración hasta llegar a 1, marcando el límite superior de la sección del array que necesita ser ordenada en esa pasada. En cada iteración, otro índice llamado i recorre los elementos desde el índice 0 hasta tam-1`, comparando cada par de elementos adyacentes. Si los elementos están en el orden incorrecto, se intercambian. Una vez que se han realizado todas las pasadas el array queda completamente ordenado.

Algoritmo (Scala):


def burbuja[A](xs: Array[A])(using ord: Ordering[A]): Unit =
  import ord.*

  def enOrden(i: Int, j: Int): Unit =
    if xs(i) > xs(j) then        // Compara elementos adyacentes
      val temp = xs(i)
      xs(i) = xs(j)
      xs(j) = temp

  for i <- 0 until xs.length - 1 do  // Bucle externo: número de pasadas
   for j <- 0 until xs.length - (i + 1) do // Bucle interno: comparaciones
     enOrden(j, j + 1)            // Llama a la comparación/intercambio

// Fin del algoritmo
