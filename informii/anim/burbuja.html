El algoritmo de la burbuja ordena un array mediante un proceso iterativo que utiliza un índice denominado tam. Este índice comienza apuntando al último elemento del array y disminuye con cada iteración hasta llegar a 1, marcando el límite superior de la sección del array que necesita ser ordenada en esa pasada. En cada iteración, otro índice llamado i recorre los elementos desde el índice 0 hasta tam-1, comparando cada par de elementos adyacentes (i e i+1). Si los elementos están en el orden incorrecto (elemento en i > elemento en i+1), se intercambian. Al final de cada pasada completa del índice i, el elemento más grande de la sección no ordenada "burbujea" hasta la posición marcada por tam. Una vez que tam llega a 1, el array queda completamente ordenado.

Algoritmo (Scala):


def burbuja[A](xs: Array[A])(using ord: Ordering[A]): Unit =
  import ord.*

  // Función auxiliar para intercambiar
  def swap(idx1: Int, idx2: Int): Unit =
    val temp = xs(idx1)
    xs(idx1) = xs(idx2)
    xs(idx2) = temp

  var tam = xs.length      // 1. Inicializar tam al tamaño total
  while tam > 1 do          // 2. Bucle externo: mientras tam > 1
    for i <- 0 until tam - 1 do // 3. Bucle interno: i de 0 a tam-2
      if xs(i) > xs(i + 1) then // 4. Comparar adyacentes
        swap(i, i + 1)      // 5. Intercambiar si necesario

    tam -= 1                 // 6. Decrementar tam para la siguiente pasada

// 7. Fin del algoritmo, array ordenado
